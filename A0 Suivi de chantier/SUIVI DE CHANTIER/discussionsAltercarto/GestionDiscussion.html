Contribuer à une discussion
Pour accéder au contenu d'une discussion, cliquez sur son titre. Cette action vous transporte dans la page de discussion. Pour participer à la discussion, créez un compte sur la page de discussion que vous avez ouverte. Ce compte vous permet de participer à toutes les discussions du site.

Format et fonctionnement des discussions
Les discussions sont ouvertes par les administrateurs du site altercarto.fr. Vous pouvez proposer un thème de discussion (contact). Le format de discussion est celui d'un chat libre. Chaque discussion est introduite par un petit texte de cadrage du thème.
