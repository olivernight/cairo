Comment naviguer dans GaïaMundi?
  • Sélectionnez un article dans le menu ci-dessus.
  • Votre sélection va déclencher l'ouverture d'un article dans le présent cadran ainsi qu'une carte dans le second cadran ( à droite).
  • Lisez le texte et cliquez sur les boutons lorsque votre lecture vous y conduit. Ces boutons déclenchent l'affichage de données dans la carte, ou bien de nouvelles cartes.

1 - Sommaire
  • Pour revenir à cette page, choisissez sommaire dans le menu ci-dessus.
  • Si vous avez besoins d'assistance cliquez sur le bouton d'aide [?] ou bien, pour une aide contextuelle, placer la souris sur le bouton ou l'objet que vous ne comprenez pas et appuyer sur la touche majuscule (shift) pour afficher l'aide ad'hoc.

(Cité Publique & altercarto [2016])
