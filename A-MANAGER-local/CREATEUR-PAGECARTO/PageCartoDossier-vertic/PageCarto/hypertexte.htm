Comment ça marche?
Lisez le texte en cliquant sur les liens bleus - Pour visualiser les données détaillées sur la carte, survolez les aires avec le pointeur de la souris - Pour visualiser le graphique d'une zone, cliquez sur la zone (vous pouvez aussi agrandir le graphique) - Pour lire les graphiques, survolez les barres avec le pointeur de la souris. Pour déplacer la carte, placez la souris sur la carte, cliquez, déplacez en maintenant le clic, puis relâchez.
