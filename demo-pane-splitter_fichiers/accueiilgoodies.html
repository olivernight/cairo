DHTML Suite for applications

This page gives you demos and documentation on some of the widgets included in DHTML Suite for applications. Demos can be opened from the menu bar above. You will also find links to standalone demos at the top ofthe description of each class (See documentation->Classes).

This page has been created by the paneSplitter class(the frames on the page) and the menuBar class(The menu at the top).

All javascript classes are located in the dhtml-suite-for-applications.js file. You will also find a copy of these classes in smaller files in the separateFiles sub folder.

Below, you can see some of the methods available for the pane splitter widget:

  • Add some external content to the west pane - Methods: addContent and showContent
  • Reload content ( The content you added to the west pane - first link - Method: reloadContent)
  • Load content ( refreshes it's self after 15 seconds - Methods: loadContent and showContent )
  • Clear refresh rate ( for the content added by the link above - Method: setRefreshAfterSeconds )
  • Delete (content with id "eastContent2" - Method: deleteContentById)
  • Delete by index (First content in east pane - Method: deleteContentByIndex)
  • Hide west pane
  • Show west pane
  • Hide east pane
  • Show east pane
  • Hide south pane
  • Show south pane
  • Hide north pane
  • Show north pane
  • New tab title of east pane (the setContentTabTitle method)
  • New title in heading - east pane ( the setContentTitle method)
  • Close all closable tabs in the center pane (the closeAllClosableTabs method)
  • Collapse west pane
  • Expand west pane

For more info, please see this Tutorial or the class documentation.
