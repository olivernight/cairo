 Bouton carte. 

Vous trouvez ce bouton dans les hypertextes. 

 

En mode hypertexte : 

Lorsque vous cliquez sur ce bouton, une carte s’affiche et les répertoires de données 
associés sont chargés dans les menus déroulants des icônes ponctuelles et des 
graphiques de données. 

 

En mode éditeur, c’est-à-dire lorsque vous fabriquez des hypertextes : 

. Cf. menu édition 
. ou bouton 


 

Le bouton carte est situé dans la barre d’outils de l’éditeur et contient la 
balise de commande d’affichage de la carte active (c’est-à-dire le dessin de 
la carte affichée, sans les données). 

 

Pour l’insérer dans l’hypertexte, placez le curseur à l’endroit où vous 
souhaitez l’insérer dans le texte, puis cliquez sur le bouton dans la barre 
d’outils. Un clone du bouton est alors placé dans le texte à l’endroit désiré. 

 

Toute cartographie de données via l’hypertexte doit commencer par l’affichage de la 
carte. Autrement dit, tout scénario de données cartographiées commence par un 
bouton rouge. 

 

Rappel : les données sont associées à des cartes. Le bouton rouge de carte ouvre en 
réalité la carte et ses données associées. Un même dessin de carte (par exemple 
France des régions) peut être associé à différents blocs de données. Ce qui vous placez 
dans l’hypertexte avec le bouton rouge, c’est une fonction qui dessine la carte et qui 
mobilise les répertoires de données associés. Ces blocs cartes + données sont 
identifiés dans le menu déroulant cartes.

 Bouton fond + icône + graphique

. Cf. menu édition dans la plateforme 
. ou Cf. bouton dans les sites filleuls

Le bouton est situé dans la barre d’outils de l’éditeur. Il contient la balise de 
commande d’affichage de la donnée représentée par des icônes ponctuelles, ainsi que 
les paramètres du graphique de données en cours dans la carte active. 

 

Pour insérer le bouton dans l’hypertexte, placez le curseur à l’endroit où vous 
souhaitez l’insérer dans le texte, puis cliquez sur le bouton dans la barre d’outils. Un 
clone du bouton est alors placé dans le texte à l’endroit désiré. 

 

 Occulte l’affichage des icônes ponctuelles en cours. Pour désinhiber l’affichage 
des icônes, cliquez une seconde fois et ainsi de suite. 

 

Attention, n’oubliez pas de désinhiber l’affichage ! Car cette inhibition persiste même 
si on change de donnée.
